title: 'Get Video-to-Task Processing Status'
sidebar_title: Get Processing Status
openapi: 'GET /task-processing-jobs/{job_id}'
---

Poll this endpoint to check the status of a video-to-task generation job using the job_id returned by the upload endpoint. See the [API Reference](/api-reference/introduction) for details on `operationId: getTaskProcessingJobStatus`.
