This guide explains how users can initiate and manage basic automation tasks through the platform interface or API.

Before diving into the details here, make sure you're familiar with the basics by reviewing the [Agent Quickstart guide](/quickstart/agent-quickstart).

Performing [Simple Tasks](/use-cases/simple-tasks) is often the quickest way to get started and see Cura in action before exploring more complex platform features like Workflows or Scheduling.
