# Use Cases: Mortgage (Real Estate Finance)

This page outlines common use cases for Curalabs within the Mortgage industry.

*   **Pre-Approval Processing:** Populate loan application data (credit, income) for quick approvals.
*   **Full Loan Origination:** Transfer all borrower/property details into LOS, generate disclosures.
*   **Title & Escrow Filing:** Submit property transfer forms, settlement docs, e-record with county offices.
