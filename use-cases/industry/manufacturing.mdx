# Use Cases: Manufacturing

This page outlines common use cases for Curalabs within the Manufacturing sector.

*   **Purchase Order Creation:** Convert inventory thresholds into orders on supplier portals.
*   **Quality & Production Logs:** Input inspection data and production run details into web-based MES systems.
*   **Invoice/Shipping Doc Matching:** Fill receiving records, reconcile vendor invoices for raw materials.
