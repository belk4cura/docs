# Use Cases: Education

This page outlines common use cases for Curalabs within the Education sector.

*   **Admissions Data Entry:** Transfer applicant info into SIS or admissions portals.
*   **Student Records Management:** Fill course registration forms, update transcripts/grades in web systems.
*   **Financial Aid & Scholarship Apps:** Populate online forms with student financial details, track eligibility.
