# Use Cases: Real Estate

This page outlines common use cases for Curalabs within the Real Estate industry.

*   **Property Listings:** Create/edit listings across MLS or third-party sites with consistent data.
*   **Tenant Screening & Leasing:** Populate rental applications, run background checks, generate lease docs.
*   **Closing Paperwork (Sales):** Transfer contract details, handle disclosure forms, coordinate e-recording.
