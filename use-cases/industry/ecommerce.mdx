# Use Cases: E-commerce

This page outlines common use cases for Curalabs within the E-commerce industry.

*   **Product Catalog Management:** Upload new product data, descriptions, images to site admin portals.
*   **Price & Stock Updates:** Bulk-edit item prices/inventory in marketplace dashboards.
*   **Order & Returns Processing:** Auto-fill return authorization forms, update order status across systems.
