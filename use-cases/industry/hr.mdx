# Use Cases: HR/Recruitment

This page outlines common use cases for Curalabs within the HR and Recruitment sectors.

*   **Job Application Screening:** Parse resumes, enter candidate details in ATS, send skill tests.
*   **Onboarding Forms (I-9, W-4):** Populate payroll/benefits enrollment, update HRIS with new hire data.
*   **Background Checks & Compliance:** Fill verification forms (criminal checks, education verifications).
