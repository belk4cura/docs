# Use Cases: Telecommunications

This page outlines common use cases for Curalabs within the Telecommunications industry.

*   **New Service Provisioning:** Gather customer/plan info, populate CRM and network portals to activate services.
*   **Order Entry & Billing Setup:** Enter plan upgrades or device changes into multiple billing/provisioning systems.
*   **Trouble Ticket/Support Form:** Fill internal support tools with customer complaints, escalate to correct department.
