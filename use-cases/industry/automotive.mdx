# Use Cases: Automotive

This page outlines common use cases for Curalabs within the Automotive industry.

*   **Credit & Financing Applications:** Copy buyer data into multiple lender web portals for loan approvals.
*   **DMV Registration & Title Transfers:** Auto-fill title forms, attach docs, submit to state systems.
*   **Warranty & Service Claims:** Enter VIN, repair details, coverage codes in manufacturer portals.
